# Phase 1 Plan 01: Foundation & Setup

## Objective

- **What:** Initialize packages, install React Router and Tailwind CSS, configure Tailwind theme variables for the dark navy/slate theme, set up client-side routing, and implement basic responsive layouts and page mount transitions.
- **Why:** Lays the foundation for all pages and components of the engineering portfolio website.
- **Output:** A compiling and running React + Vite + Tailwind + React Router application with base layout and dark mode theme.

<task type="code">
  <name>Configure Tailwind CSS & CSS Variables</name>
  <files>tailwind.config.js;src/index.css</files>
  <read_first>
    - tailwind.config.js
    - src/index.css
  </read_first>
  <action>
    - Update `tailwind.config.js` to scan `./index.html` and `./src/**/*.{js,ts,jsx,tsx}`.
    - Extend the theme in `tailwind.config.js` to support the portfolio colors (e.g., custom dark background colors, cyan accent, violet accent).
    - Update `src/index.css` to import Tailwind's base, components, and utilities.
    - Define CSS custom variables under `@theme` or `:root` in `src/index.css` for the dark mode theme:
      - `--bg-dark-primary`: `#0B0F19`
      - `--bg-dark-secondary`: `#0F172A`
      - `--color-accent-primary`: `#38BDF8` (cyan)
      - `--color-accent-secondary`: `#818CF8` (violet)
      - `--color-text-primary`: `#F8FAFC` (pure white)
      - `--color-text-secondary`: `#E2E8F0` (off-white body text)
  </action>
  <acceptance_criteria>
    - `tailwind.config.js` contains content array mapping index.html and src directory
    - `src/index.css` contains `@tailwind base`, `@tailwind components`, and `@tailwind utilities`
    - `src/index.css` contains custom variables for dark slate/navy theme
  </acceptance_criteria>
  <verify>Check file contents of tailwind.config.js and src/index.css.</verify>
  <done>Tailwind CSS is configured and index.css has the theme variables defined.</done>
</task>
